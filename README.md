# 📚 LibraryHub

A full-stack library management system built with the **MERN Stack** that enables users to manage books efficiently through a clean and responsive web interface.

---

## 🚀 Features

* 📖 Add new books
* ✏️ Update existing book details
* 🗑️ Delete books
* 🔍 Search books by title
* 🏷️ Filter books by category
* 📚 View all books in a structured layout
* 💾 Store and manage data using MongoDB
* 📱 Responsive user interface

---

## 🛠️ Tech Stack

### Frontend

* HTML5
* CSS3
* EJS

### Backend

* Node.js
* Express.js

### Database

* MongoDB
* Mongoose
